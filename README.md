## Mess with it locally
You will Need:

  -Local MongoDb server running.
  
  -Node and Npm.
  
  -Angular Cli (npm install -g @angular/cli)
  
  from '/'
  
run `npm install`

Run `node server.js'

Run 'npm start' for your local version on port 4200 :v:
